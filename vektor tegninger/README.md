# Vektor tegninger

Denne mappen inneholder en enkel vektorversjon av plantegningen i wayfinding-stil.

## Filer
- `amfi-steinkjer-enkel-vektor.svg` – forenklet kart med:
  - butikk-/sonesegmenter
  - markert gangareal
  - inngangspunkter (POI)
  - eksempelrute (start → mål)

## Bruk i prosjektet
- Kan åpnes/redigeres i Figma, Illustrator, Inkscape eller Super Figure.
- Kan eksporteres til PNG/WebP og brukes som bakgrunn i `screens.json`.
- Om du vil, kan vi neste steg splitte kartet i etasjer/tema (f.eks. `map1`, `map2`, `tech_map1`) og lage hotspot-koordinater direkte.

## Viktig
Tegningen er en bevisst forenklet versjon for wayfinding-UI (ikke målfast CAD-tegning).
